# Task API

A simple CRUD API built with Python and FastAPI.

The API supports creating, reading, updating, and deleting tasks.

Each task contains:

- `id` — unique task ID
- `title` — task title
- `done` — completion status

## Requirements

- Python 3.10+
- FastAPI
- Uvicorn
